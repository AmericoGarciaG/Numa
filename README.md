# Numa

## About The Project

Numa es un asistente financiero personal que elimina la fricción de la gestión financiera a través de la conversación y la IA.

## Getting Started

### Prerequisites

- Python 3.9+
- Git

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/AmericoGarciaG/Numa
   ```
2. Create and activate a virtual environment
   ```sh
   python3 -m venv .venv
   source .venv/bin/activate
   ```
3. Install Python packages
   ```sh
   pip install -r requirements.txt
   ```
